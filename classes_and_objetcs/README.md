# ☕ Fundamentos de Java — Classes e Objetos

Este diretório contém programas simples em Java desenvolvidos durante o aprendizado de Programação Orientada a Objetos (POO).

O foco desses projetos é praticar:

* Criação de **classes**
* Instanciação de **objetos**
* Definição e uso de **métodos**
* Manipulação de **atributos**
* Encapsulamento básico

## 📂 Projetos

### 🔹 Employee Salary Raise

Simula um funcionário com nome, salário bruto e imposto.

O programa:

* Calcula o salário líquido
* Exibe o status do funcionário
* Aplica um aumento percentual no salário bruto

### 🔹 Pen

Simula o comportamento de uma caneta.

O programa:

* Define características como modelo, cor, ponta e carga
* Mostra o estado atual da caneta
* Permite tampar e destampar
* Verifica se é possível escrever (dependendo se está tampada)
